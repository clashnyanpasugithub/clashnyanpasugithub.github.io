2.2 动态难度副本

"深渊回廊"副本采用AI自适应的"噬魂"机制：队伍DPS越高，怪物获得的免伤加成越强，但会相应降低控制抗性。这迫使玩家必须平衡输出与控制链的组合，某次实测中，采用3法师+1战士的极端阵容反而比均衡阵容通关速度快27%。

2.3 装备词缀联动

FAQ

PassWall2 如何根据延迟自动切换节点？
PassWall2 会定期测试节点延迟，并结合策略组自动切换到延迟最低的节点。无需用户手动操作，即可保证网页、视频和游戏流量的高速稳定访问，提高整体网络体验。
Mellow TUN 模式与 SOCKS5 代理有什么区别？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响指定应用。TUN 更适合全局加速，如游戏、视频和浏览器流量，提高网络稳定性和覆盖范围。
SSR 的 UDP 转发功能如何启用？
SSR 支持 UDP 转发，可通过客户端设置开启。启用后，游戏和语音等 UDP 流量可通过节点代理，降低丢包率和延迟，提高实时网络应用体验。
SSR 配置如何提高节点隐蔽性？
通过选择合适协议和混淆方式，将流量伪装成正常 HTTPS 或 HTTP 数据。可降低被 DPI 检测和封锁的风险，提高节点长期稳定性和科学上网体验。
Surfboard 支持按应用分流流量吗？
支持。Surfboard 可通过策略组和规则文件将不同应用的流量分配到指定节点，保证游戏、视频和浏览器流量走最优节点，提高访问速度和网络稳定性。
Hysteria2 在丢包环境下如何配置更稳？
可调整 Hysteria2 的拥塞控制与重传参数，启用 FEC（前向纠错）并合理设置 MTU 与窗口大小以适应链路特性。还可尝试切换 QUIC 参数或更换中转以减少丢包影响。
Sing-box 如何按 IP 段实现智能分流？
Sing-box 支持根据 IP 段匹配流量，将国内流量直连、国外流量走代理。结合策略组和规则文件，实现精细化流量管理，保证访问速度和稳定性，同时提高节点使用效率。
Matsuri 项目是什么？
Matsuri 是基于 sing-box 的跨平台 GUI 客户端，界面现代，支持多种协议和分流方式。它提供直观的节点管理和日志监控功能，适合喜欢图形化操作的用户。Matsuri 的设计目标是简洁、高效、易于上手。
Nekobox 是否支持自定义 DNS 解析？
支持。用户可以在配置中指定 DNS 服务器或启用 DoH/DoT 功能，确保域名解析安全和快速，避免本地 DNS 污染或被劫持。
Hiddify Next 的多用户限速功能如何使用？
Hiddify Next 支持在管理面板中为每个用户设置带宽上限。管理员可以按需分配速率或流量配额，防止单一用户占用过多资源，从而保持整体网络稳定。
