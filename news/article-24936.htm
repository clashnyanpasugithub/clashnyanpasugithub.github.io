java list在哪个包_Java 中初始化 List 集合的 N 种方式！

List 是 Java 开发中经常会使用的集合，你们知道有哪些方式可以初始化一个 List 吗？这其中不缺乏一些坑，在此总结一下常用的方式。

1、常规方式List languages = new ArrayList<>();

languages.add("Java");

languages.add("PHP");

languages.add("Python");

System.out.println(languages);     // [Java, PHP, Python]

这种就是我们平常用的最多最平常的方式了，没什么好说的，后面缺失的泛型类型在 JDK 7 之后就可以不用写具体的类型了，改进后会自动推断类型。

2、Arrays 工具类List jdks = Arrays.asList("JDK6", "JDK8", "JDK10");

System.out.println(jdks);   // [JDK6, JDK8, JDK10]

这种方式添加的是只读的，是不可变的 List，即不能添加、删除等操作，需要警惕。。

如果要可变，那就使用 ArrayList 再包装一下，如下面所示。List numbers = new ArrayList<>(Arrays.asList("1", "2", "3"));

numbers.add("4");

System.out.println(numbers);

包装一下，这就是可变的 ArrayList 了。

3、Collections 工具类List apples = Collections.nCopies(3, "apple");

System.out.println(apples); // [apple, apple, apple]

这种方式添加的同样是只读的，是不可变的。

老规则，如果要可变，使用 ArrayList 包装一遍。List dogs = new ArrayList<>(Collections.nCopies(3, "dog"));

dogs.add("dog");

System.out.println(dogs);

还有初始化单个对象的 List 工具类，这种方式也是不可变的，集合内只能有一个元素，这种也用得很少啊。List cat = Collections.singletonList("cat");

System.out.println(cat);

还有设置集合为只读的。使用 unmodifiableList 方法。// 使用 Collections.unmodifiableList 创建只读集合

List list = new ArrayList<>();

list.add("jack");

list.add("tom");

list.add("lily");

List readList = Collections.unmodifiableList(list);

4、匿名内部类List names = new ArrayList<>() {{

add("Tom");

add("Sally");

add("John");

}};

System.out.println(names);

这种使用了匿名内部类的方式，一气喝成，是不是很高大上？看似高级，实现也没什么卵用。

5、JDK8 StreamList colors = Stream.of("blue", "red", "yellow").collect(Collectors.toList());

colors.add("beautiful");

System.out.println(colors);     // [blue, red, yellow, beautiful]

Stream 是 JDK 8 推出来的新概念，比集合还要更强大，还可以和集合互相转换。

6、JDK 9 List.ofList cups = List.of("A", "B", "C");

System.out.println(cups);

这是 JDK 9 里面新增的 List 接口里面的静态方法，同样也是不可变的。

7、Guava 工具包// 使用 Lists.newArrayList 创建的集合非只读

ArrayList list = Lists.newArrayList("java", "php", "javaScript", "h5", "jackson");

// 使用 ImmutableList.of 创建只读集合

ImmutableList list = ImmutableList.of("jack", "tom", "lily");

说到最后，还是 Guava 用的最爽，对于集合的控制也很多。
